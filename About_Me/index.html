About Me!

I'm Nexint, a YouTuber, coder, gamer and musician! I currently run a YouTube channel and code some small projects.

I use FL studio to make my music, which is a powerful music editing program that I paid for, in conjunction with a keyboard and mouse to make this possible.

I use Davinci resolve in conjunction with OBS to record and edit our videos. I also use sound effects and some music from Pixabay, and make thumbnails using BeFunky.

I use VS Code to code for most real projects, including this website and DDLT (dead). However, on Batch specifically, I prefer using notepad, and on Scratch I use the default editor.

Our main setup is a fairly decent setup, with a keyboard, mouse, 4K 60FPS monitor, and speakers.
This PC has these specs:
- i7 12700KF (Stock)
- RTX 3070 (Stock)
- 32gb DDR4 (3200Mhz) (XMP)
- 1tb NVME + 1tb SSD + 6tb HDD + 5tb HDD (Combined: 2tb SSD, 11tb HDD)

Our alternate setup is generally average, with a laptop keyboard, mouse, 1080p laptop monitor, and a headset.
This laptop has these specs:
- i5 7300HQ (Stock)
- GTX 1050 Mobile (Stock)
- 12gb DDR4 (2400Mhz)
- 256gb NVME + 1tb HDD
